## General Method

How to train your model?

- Pre-processing
    - numerical vs categorical (e.g. categorical bad for regression)
    - categorical cardinality (e.g. what if 200 category?)
    - encoding
    - null
    - distribution/imbalance
    - noise/outlier
    - correlation (e.g. heatmap)
    - feature importances
- Train
    - select model
    - standard scaler
    - train/test split
    - fit & predict
    - tuning
- Evaluation
    - regression (r2)
    - classification (accuracy_score, f1, AUC)
